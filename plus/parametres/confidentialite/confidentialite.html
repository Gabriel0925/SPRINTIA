Confidentialité

Dernière mise à jour de la politique de confidentialité : 23 Mai 2026

Le but de SPRINTIA n'est pas de collecter des données, mais de vous aider avant et après vos entraînements. C'est pour cela que SPRINTIA reste transparent sur les données utilisateur. Je vous explique tout dans cette politique de confidentialité.

Données personnelles

  • Première chose, je ne collecte aucune donnée personnelle à votre insu. SPRINTIA stocke uniquement les données nécessaires au fonctionnement des algorithmes et uniquement quand vous cliquez sur un bouton de sauvegarde.
  • Deuxième chose, les données que vous enregistrez sont stockées dans votre navigateur, ça veut dire que moi-même qui suis le développeur de SPRINTIA, je n'ai pas accès à vos données. Étant donné que je n'ai pas accès à vos données vu qu'elles restent stockées dans votre appareil, je ne peux donc pas les vendre aux GAFAAM (Google, Apple, Facebook, Amazon, Microsoft) ou autres.

Point d'attention

Les données étant stockées dans votre navigateur, si vous effacez le cache de votre navigateur ou que vous changez d'appareil ou de navigateur, SPRINTIA aura perdu la mémoire car vos données auront été entièrement supprimées.

Pour y remédier, SPRINTIA possède une fonctionnalité de sauvegarde et de restauration accessible ici. C'est pour cela que je vous conseille d'essayer de télécharger vos données de temps en temps, au moins si vous avez perdu vos données par erreur ou autre, vous pourrez les restaurer grâce à ce fichier de sauvegarde.

Je sais que ce n'est pas le plus intuitif mais ça permet à SPRINTIA d'être respectueux de vos données personnelles et ça permet aussi à SPRINTIA de ne pas vous obliger à créer un compte pour pouvoir l'utiliser. Si je trouve un moyen de simplifier le processus je le ferai sans hésitation mais pour le moment je n'en ai pas vraiment malheureusement.

Statistiques de navigation

J'utilise une API de Vercel (Vercel Web Analytics) pour obtenir des statistiques, ce qui me permet d'améliorer SPRINTIA en voyant quelles fonctionnalités sont les plus utilisées par les utilisateurs de SPRINTIA.

Un point important, Vercel ne collecte pas les données que vous enregistrez dans SPRINTIA tout simplement parce qu'il n'y a pas accès. Comme dit précédemment, vos données restent sur votre appareil.

Vercel collecte uniquement de manière totalement anonyme, les pages que vous consultez pour me donner des statistiques sur les pages les plus vues au quotidien, le type d'appareil utilisé ainsi que le pays où vous vous situez (pas votre localisation précise, uniquement le pays). Je n'ai également aucun accès à votre adresse IP.

Cette API Vercel est fournie par défaut et je décidé de l'utiliser car cette API ne met pas en danger vos données personnelles, elle me permet uniquement d'avoir quelques stats sur SPRINTIA. Cette API fonctionne sans aucun cookie et respecte le RGPD.
