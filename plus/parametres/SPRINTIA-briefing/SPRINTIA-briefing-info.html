À propos de SPRINTIA Briefing

Quelles données sont partagées ?

SPRINTIA Briefing génère un prompt qui contient différentes données en fonction de l'endroit où SPRINTIA Briefing est présent. Par exemple, dans la charge d'entraînement vos données d'entraînement sont envoyées mais dans l'indulgence de course c'est uniquement vos données de course qui sont envoyées, ça dépend vraiment du but de l'algorithme.

Néanmoins, voici tout ce que SPRINTIA peut intégrer dans le prompt :

  • Vos données d'entraînement (des 30 derniers jours maximum) : date, sport, distance, durée, RPE, charge d'entraînement,...
  • Vos données de récupération : FC Repos des 7 derniers jours.

Comment les données sont transférées ?

En cliquant sur le bouton "Copier & Ouvrir [votre IA préférée]" (par défaut Vibe), vos données ne sont pas envoyées à l'IA directement. En réalité, le prompt est copié dans votre clavier et SPRINTIA Briefing vous dirige dans un nouvel onglet vers votre IA préférée. Une fois sur l'application de votre IA, vous n'avez plus qu'à coller ce prompt et si vous appuyer sur le bouton "Envoyer" de votre IA c'est à ce moment que l'IA va connaître vos données d'entraînement,...

Recommendations

Je documente ce process car SPRINTIA stocke toutes vos données en local, ce qui permet d'avoir une excellente sécurité des données. Mais SPRINTIA Briefing récupère vos données pour les mettre dans un prompt et une fois que vous les envoyez à votre IA préférée, vos données ne sont plus en local sur votre appareil. Les IA vont sûrement s'entraîner sur vos données pour je cite "Améliorer leurs produits au bien de tous". Pour pallier cette contrainte, j'ai plusieurs solutions :

  • Premièrement si vous êtes sur Android/IOS, vous pouvez télécharger l'application "Google AI Edge Gallery". Cette application est géniale car elle permet de télécharger des modèles d'IA sur votre appareil ce qui signifie que vos données restent sur votre appareil ! Je vous recommande d'installer le modèle nommé "Gemma-4-E2B-it".
  • Deuxièmement si vous êtes sur ordinateur, vous pouvez utiliser une application nommée "Ollama" pour télécharger des modèles d'IA qui tournent en local sur votre ordinateur, ce qui permet de garder toutes vos données bien au chaud sur votre machine !

Pourquoi l'IA par défaut de SPRINTIA Briefing est Vibe ?

J'ai choisi de mettre Vibe (anciennement le chat de mistral) car c'est une excellente IA française pour analyser des données et étant donné que SPRINTIA a été créé en France ça me tient à coeur d'utiliser des services qui sont Français ! Toutefois, si vous le souhaitez vous pouvez choisir l'IA que vous utilisez au quotidien depuis cette page.
