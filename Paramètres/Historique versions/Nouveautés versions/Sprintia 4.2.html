Sprintia 4.2

Information

Date de sortie : 19 Avril 2026
Taille du site : ... Mo

Sprintia 4.2 apporte la possibilité d'importer vos données d'entraînement depuis Coros, Garmin, Strava et plus encore. Tout ça grâce à un fichier (.FIT) ce qui est beaucoup plus fiable qu'avant et qui permet d'importer plus de données. De plus, Sprintia Labs fait son apparition pour entraîner mes algorithmes futurs ainsi que pour tester toutes mes idées.

🆕 Nouvelles fonctionnalités

  • Désormais, vous pouvez partager vos entraînements en image que vous pouvez mettre par la suite en story Insta,... ❌
  • Vous pouvez enfin importer vos données d'entraînement depuis n'importe qu'elle plateforme grâce à un fichier (.FIT). De plus, au lieu d'avoir à saisir l'entraînement que vous venez de faire, grâce à cette fonctionnalité vous pouvez désormais importer le fichier de votre séance dans Sprintia pour l'importer avec toutes vos données (FC, Distance,...) ce qui va vous faire gagner du temps.
  • Dans Sprintia 4.2 vous pouvez enregistrer si vous le souhaitez vos données personnelles (qui sont sauvegardé en local sur votre appareil) comme par exemple votre Genre, Poids, FC Repos et votre FC Max. Ces données ne sont pas obligatoire mais elles permettent de débloquer des fonctionnalités comme l'estimation des calories brûlées lors de vos entraînements,...
  • C'est l'arrivée de Sprintia Labs, un espace dans les paramètres qui me permet de tester mes idées pour les futurs algorithmes de Sprintia. De plus, c'est également dans cet espace que je vais entraîner mes algorithmes a être le plus précis grâce à votre utilisation de ce dernier. Par conséquent, plus vous utilisez Sprintia Labs, plus mes futures algorithmes seront précis.

📊 Améliorations

  • Le responsive de tous les écran a été retravaillé pour l'optimiser, le rendre plus pertinent peu importe l'appareil mais également réduire la quantité de code qui était inmaintanable.
  • Une nouvelle interface fais son apparition dans les paramètres, tout est mieux trier et rangé.
  • Ajout d'un raccourci lors de l'ajout d'un entraînement qui permet d'importer un fichier (.TCX) de votre entraînement pour l'ajouter rapidement dans Sprintia.
  • L'import via TrainingPeaks refais surface, donc vous pouvez importer votre historique d'entraînement grâce à un fichier (.CSV) que vous pouvez télécharger depuis TrainingPeaks. En plus de ce retour, l'import via TrainingPeaks a été amélioré pour être plus fiable et pour importer plus de données que l'ancien import comme la cadence moyenne, la FC moy., la FC max., la description de vos entraînements et de nouveaux sports sont importable comme : le ski, le rameur d'intérieur et la randonnée.
  • Désormais, une fois que vous avez importer votre entraînement ou votre historique d'entraînement, ça vous envoie directement sur la page d'historique d'entraînement.
  • Le bouton 'ajouter un entraînement' flotte en bas à droite de l'interface pour être plus facilement accessible. De plus, il est plus gros pour montrer son importance.

🐛 Correction des bugs et optimisations

  • Ajout de plusieurs cursor pointer dans toutes la PWA de Sprintia.
  • Grosse optimisation des classes CSS. De plus, la maintenabilité du fichier CSS a été grandement améliorée.
  • L'icône 'supprimer' a été supprimée de l'historique des statuts.
  • Amélioration de la gestion des erreurs dans l'outil nommé "Niveau de course".
  • Ajout d'une ombre portée ainsi que de nouvelles couleurs d'écriture pour les notes d'entraînement pour plus coller au design des inputs dans Sprintia.
  • Optimisation majeure de la page 'Thèmes' ainsi que de la page historique des versions.
  • Correction de plusieurs erreurs d'orthographe.
  • Correction d'un bug : la vitesse maximum n'était pas arrondit lors de l'affichage.
  • Le code du JRM Coach sur la page d'accueil est beaucoup plus sécurisé. Par conséquent, le risque de plantage est fortement réduit.
  • Une nouvelle fonction globale à Sprintia fais son apparition ce qui permet d'avoir accès aux 30 derniers jours d'entraînements de l'utilisateur ce qui optimise le code de l'application car tout se retrouve au même endroit.❌
  • La fonctionnalité de restauration est désormais bien plus sécurisé, de plus le code risque de moins planter car il y a désormais plus de vérification.
  • L'import de votre historique d'entraînement depuis un fichier (.CSV) Garmin est désormais plus fiable et plus sécurisé.
  • Amélioration du responsive du header et ajustement de l'animation du logo dynamique.
