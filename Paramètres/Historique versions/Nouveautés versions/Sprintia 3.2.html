Sprintia 3.2 - Votre coach

Information

Type : Mise à jour mineure
Date de sortie : 23 Octobre 2025

Sprintia 3.2 introduit un coach entièrement personnalisable. Vous pouvez choisir son nom, son style et même son avatar. Votre coach, à votre image, pour te motiver au quotidien !

🆕 Nouvelles fonctionnalités

  • La fonction VMA fait peau neuve :
    1. Nouvelle interface : organisation en cadres,...
    2. Plusieurs tests disponibles : demi-Cooper, Cooper, Luc Léger, course de référence.
    3. Tableau des zones enrichi avec allures, objectifs et exemples de séances.
    4. Texte explicatif sur la VMA et son utilité.
    5. Précision améliorer grâce à une nouvelle méthode de calcul pour le test : "Course de référence"
  • Auto-connect : plus besoin de vous reconnecter à chaque lancement de Sprintia, vous restez connecté automatiquement.
  • Votre Coach vous donne vos statistiques d'entraînement des 7 derniers jours.
  • Les onglets : objectif et compétition mémorise désormais votre choix entre "Objectif futur" et "Objectif passé".
  • Vous pouvez désormais modifier vos objectifs/compétitions passées.
  • Lors de la création de votre compte, vous n'avez plus le droit de choisir des mots jugés sensibles, inappropriés.
  • Remplissage automatique de certains champs lors d'un enregistrement d'un entraînement, on l'aperçoit grâce à cette icône "💡".
  • Nouveau mode 'Libre' quand vous ajoutez un entraînement : les modes 'Extérieur' et 'Intérieur' se sont assemblés pour former le mode 'Libre' :
    Le mode 'Libre' peut-être utilisé si ton entraînement ne correspond pas aux modes 'Course', 'Musculation' ou 'Football'.
  • Arrivé de votre coach de sport (JRM Coach) qui est intégré à plusieurs endroits dans Sprintia :
    1. Il peut vous donner des conseils, des infos, des phrases de motivation, et des tips sur Sprintia.
    2. Vous pouvez personnaliser le nom de ce coach.
    3. Vous pouvez choisir un style de coach (bienveillant, strict mais motivant, pote, inshape). 4. Vous pouvez même choisir son avatar !
  • Une nouvelle option : "Autre" est disponible pour la priorité de la compétition.
  • Une nouvelle option : "Étirement" est désormais disponible pour le type d'entraînement (dans le mode libre).
  • De nouvelles options : "Futsal", "Autre" sont maintenant disponible pour le type d'entraînement (dans le mode football).
  • De nouvelles options : "Gymnase", "Street Workout", "Extérieur", "Autre" sont disponible pour le lieu de l'entraînement (dans le mode musculation).
  • Une nouvelle option : "Mixte" est disponible pour le type d'entraînement (dans le mode musculation).
  • De nouvelles options : "Seuil", "Récupération", "Fun" sont désormais disponible pour le type d'entraînement (dans le mode course).
  • Nouvel outil nommé "Protéines quotidiennes" :
    Cet outil vous indique la quantité optimale de protéines à consommer chaque jour. Grâce à ton poids, ton niveau d'activité (sédentaire, sportif d'endurance,...).

📊 Améliorations

  • Ajout d’une boîte de dialogue de confirmation avant l’ouverture d’app externe.
  • Ajustement de l'interface utilisateur pour améliorer l'UX et l'UI.
  • Simplification et amélioration de l'ergonomie pour la modification de votre compte.
  • Plus besoin d'éditeur de code pour lancer Sprintia, désormais vous n'avez plus qu'à double-cliquer sur l'app "Sprintia.exe".
  • Vous pouvez désormais créer un raccourci sur votre bureau pour lancer Sprintia plus rapidement et avec une icône.
  • La longueur de votre bio est désormais limitée à 80 caractères pour une meilleure lisibilité.
  • Nouveau style de tableaux pour une meilleure lisibilité.
  • Simplification des statuts de charge d'entraînement désormais il n'y plus que ("Sous-entraînement", "Progression optimale" et "Surentraînement").
  • Les boutons s’adaptent désormais pour occuper l’espace disponible.
  • Ajout de l'unité de l'IMC.
  • Les pop-ups d'aide ont été supprimées pour éviter de gêner l'utilisateur.
  • Vous pouvez compléter les lignes des champs déjà rempli sans tout effacer (lors de la modification d'un objectif, compétition).
  • Suppresion de l'accès aux actualités de Sprintia parce que rien n'était posté.
  • Demande de l'ancien mot de passe lors de la modification du mot de passe.
  • De nouveaux caractères spéciaux sont prises en charge pour les mots de passe.
  • Désormais, la valeur de RPE par défaut est 1.
  • Les mots ne sont plus coupé dans les 'TextBox'.
  • Limite de caractère lors de l'enregistrement de données (ex : ajouter une activité, ajouter un objectif,...).
  • Ajout de plus de 10 choix de type d'entraînement pour plus que l'utilisateur soit libre de choisir le type d'entraînement qu'il souhaite.
  • Ajout d'un calendrier pour choisir la date quand vous ajoutez un entraînement.
  • Tous les formats de date sont acceptés (JJ-MM-AAAA, JJ/MM/AAAA, JJ.MM.AAAA, JJ MM AAAA, JJ_MM_AAAA, JJ,MM,AAAA).
  • Formats de temps élargis (hh:mm:ss, mm:ss, minutes).
  • Quand vous saisissez une distance, vous pouvez utiliser des virgules, des espaces, des tirets bas, "km" ou "KM" (ex : 10,5 km, 10 5, 10_5, 10.5).
  • Sprintia s'ouvre directement en plein écran pour vous faire gagner du temps.
  • Réorganisation des options pour les combobox pour paraître pour cohérent aux yeux des utilisateurs.
  • Désormais, la touche "Échap" permet de quitter le mode plein écran.
  • Design unifié, peu importe si votre ordinateur est en mode sombre ou clair, les couleurs ne s'adapteront plus, cela permet de renforcer l'identité visuelle de Sprintia.

🐛 Correction des bugs et optimisations

  • Cliquer sur la croix d'une boite de dialogue ferme désormais la boîte de dialogue, au lieu d’ouvrir quand même l’application (ex : le google drive des actus de Sprintia).
  • Correction de fautes d'orthographe dans l'application.
  • Optimisation du code pour améliorer la maintenance.
  • Amélioration de la sécurité.
  • Amélioration de la cohérence de Sprintia.
  • Code désormais bien plus robuste.
  • Correction d'un bug lors de la modification d'un objectif, compétition.
  • Correction d'un bug qui empêchait la mise en pause des analyses.
  • Correction d'un bug majeur qui ne supprime pas un entraînement en fonction du mode d'entraînement.
  • Optimisation de l'interface en fonction de la taille de l'écran.
  • Correction d'une erreur qui se produisait lors de la validation par la touche "Entrée" pendant les transitions d'interface.
  • Correction d'un bug qui empêchait le retour à la ligne dans 'Contribue à améliorer Sprintia'.
  • Amélioration des vérifications d'enregistrement d'entraînement pour éviter les erreurs ou les bugs.
  • Amélioration des vérifications pour la suppression d'un entraînement, d'un objectif ou d'une compétition.
  • Amélioration des vérifications de calcul dans les outils pour éviter les erreurs ou les bugs.
  • Correction d'un bug qui empêchait de valider avec la touche "Entrée" dans ajouter un objectif.
  • Correction d'un problème d'affichage mineur dans la charge d'entraînement, certains blocs dépassaient légèrement de son conteneur.
  • Refonte back-end : découpage du gros fichier en plusieurs fichier pour un code plus organisé, lisible et maintenable.
