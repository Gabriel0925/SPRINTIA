Sprintia 4.3.2

Correction de bugs

Date

06 Juin 2026

Taille

3,11 Mo

Résumé

Sprintia 4.3.2 apporte de nombreuses corrections de bugs majeurs qui sont liés à la dernière mise à jour qui avait pour but d'optimiser et de corriger les bugs ce qui est un peu ironique car au final cette mise à jour a apporté des bugs majeurs.

Patch note

Correction des bugs et optimisations

  • Correction d'un bug dans la page de statistiques : lorsque l'utilisateur n'avait pas de données sur les 7 derniers jours et que l'utilisateur voulait afficher les statistiques des 30 derniers jours, le graphique ne réapparaissait pas car il avait été caché auparavant c'est désormais corrigé.
  • Correction d'un bug dans la page de récupération : lorsque l'utilisateur n'avait pas de données de récupération sur les 7 derniers jours le bouton pour voir l'historique complet disparaissait ce qui empêchait l'utilisateur d'accéder à son historique complet des dernières semaines, ce bug est désormais corrigé.
  • Correction d'un bug du bouton de réinitialisation du thème : lorque l'utilisateur cliquait sur le bouton de réinitialisation je passais le mauvais id du thème par défaut (car les id des éléments on changé dans la v4.3.1) ce qui faisait que le thème se chargeait mais aucun thème était séléctionné sur la page, c'est désormais corrigé.
  • Ajout d'un arrondi au centième pour le graphique de l'indulgence de course.
  • Optimisation de la requête à la base de données pour le graphique de volume d'entraînement sur la page entraînement.
