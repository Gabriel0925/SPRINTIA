Importer mes séances

Garmin

Vous avez une montre Garmin et vous voulez importer votre historique d'entraînement sur Sprintia, vous êtes au bon endroit. Pour importer vos données Garmin, veuillez télécharger le fichier CSV de votre historique d'entraînement disponible sur Garmin Connect Web. Pour récupérer ce fichier, suivez les instructions :

  • Cliquez sur le lien ci-contre.
  • Connectez-vous à votre compte Garmin.
  • La langue de Garmin Connect Web doit être le Français ou l'Anglais.
  • Cliquez sur "Export CSV" (en haut à droite), le fichier CSV a dû se télécharger.

TrainingPeaks

Vous avez synchronisé votre montre (Apple Watch, Coros, Polar, Suunto...) avec TrainingPeaks. Vous pouvez grâce à cela importer votre historique d'entraînement sur Sprintia en important le fichier '.csv' que vous pouvez trouver sur le site web de TrainingPeaks. Pour ce faire, suivez les instructions :

  • Cliquez sur le lien ci-contre.
  • Connectez-vous à votre compte TrainingPeaks.
  • Allez dans les paramètres (cliquer sur votre photo de profil en haut à droite puis "Settings").
  • Rendez-vous dans "Export data".
  • Remplissez les champs de la catégorie "Workout Summary".
  • Appuyez sur "Exporter".

Pour finir, vous n'avez plus qu'à dézipper le dossier et importer le fichier CSV nommé 'workouts.csv' dans Sprintia.

Runkeeper

Vous utilisez Runkeeper quand vous courez ou vous avez connecté votre montre connectée à Runkeeper, dans les 2 cas vous pouvez télécharger un fichier (csv) de votre historique d'entraînement depuis le site web de Runkeeper pour pouvoir l'importer dans Sprintia. Pour ce faire, suivez les instructions :

  • Cliquez sur le lien ci-contre.
  • Connectez-vous à votre compte Runkeeper.
  • Remplissez les champs de date.
  • Cliquez sur "Export".

Pour finir, vous n'avez plus qu'à dézipper le dossier et importer le fichier CSV nommé 'cardioActivities.csv' dans Sprintia. Sprintia a besoin de votre âge pour estimer le RPE de chaque séance du fichier CSV Runkeeper.
