SPRINTIA 4.0.2 : Corrections majeures

Information

Type : Patch majeur
Date de sortie : 07 Mars 2026
Taille du site : 2,44 Mo

SPRINTIA 4.0.2 apporte de nombreuses optimisations et corrections de bugs ce qui améliore la stabilité de SPRINTIA. De plus, SPRINTIA 4.0.2 vous offre la possibilité d'importer votre historique d'entraînement depuis un fichier CSV qui provient de Garmin Connect (Web), TrainingPeaks ou encore Runkeeper.

🆕 Nouvelles fonctionnalités

  • Vous pouvez désormais importer la totalité de votre historique d'entraînement depuis un fichier csv sur le site de Garmin, TrainingPeaks ou encore Runkeeper.
  • Optimisation de l'analyse de la charge d'entraînement et de l'indulgence de course : Intégration d'un algorithme de calibrage progressif qui permet d'avoir une bonne fiabilité dès la première semaine d'entraînement.

📊 Améliorations

  • Changement de design pour les listes à puce.
  • L'indulgence de course conseille désormais une plus grande fourchette par exemple sur SPRINTIA 4.0.1 => 28,5 - 29,3 km, désormais sur SPRINTIA-4.0.2 => 28,0 - 29,3 km.
  • 2 nouveaux avatars de coach sont disponibles : "🏃", "🚴".
  • Nouvelle interface dans la page d'accueil, avec un bouton "ajouter" repensé et un statut plus mis en avant.
  • Il y a de nouveau l'effet de rebond lors du clic sur une carte.
  • Une nouvelle licence fais son apparition ce qui protège mon code pour éviter que quelqu'un utilise mon code à but commercial ou pour créer une société. De plus, un utilisateur n'a pas le droit de prendre mon code, de changer le logo, et de le remettre en ligne sous son propre nom (même gratuitement).
  • Désormais, l'ensemble des outils de SPRINTIA se base sur des études scientifiques.
  • Le menu de navigation (=burger-menu) se referme automatiquement lors d'un scroll sur la page web.
  • Ajout d'un raccourci pour aller rapidement à son historique d'entraînement, désormais vous pouvez cliquer sur l'interprétation JRM de votre semaine d'entraînement (sur la page d'accueil), ça vous ammenera à votre historique d'entraînement.
  • Ajout d'une ombre sur l'icone de SPRINTIA.

🐛 Correction des bugs et optimisations

  • Correction d'un bug qui empêchait le calcul de vos Zones cardiaques avec la méthode FC max.
  • Correction d'un bug qui empêchait l'affichage de certaines icônes dans les cartes des outils.
  • Correction d'un bug qui permettait d'afficher 24.00330033030 min dans le JRM Coach sur la page d'accueil.
  • La distance est désormais arrondie dans l'historique d'entraînement pour éviter de se retrouver avec 2,677777777 km.
  • Correction d'un bug de la charge d'entraînement qui bloquait les analyses si vous n'aviez pas rentré d'entraînement depuis 7 jours.
  • La cohérence des textes à l'intérieur des boutons quand une action se lance a été améliorée.
  • Correction d'un bug très mineur dans l'outil : "Niveau de course".
  • Les coins sont moins arrondis et désormais ils respectent la loi d'UI de continuité des bordures pour l'ensemble des éléments d'interface de SPRINTIA.
  • Le code couleur des messages d'erreur dans l'outil Niveau de Course respecte désormais le code couleur imposé par la mise à jour SPRINTIA 4.0.1.
  • Grosse optimisation du code CSS avec de meilleurs noms de class,...
  • Le rythme des animations a été ajusté pour être plus lent au début de l'animation puis plus rapide à la fin de l'animation.
  • Correction d'un bug dans le JRM coach qui affichait : "JRM Coach ::".
  • Correction d'un bug lors de la modification d'un entraînement avec 0 m de dénivelé.
  • L'effet "tap highlight color" a été supprimé de l'entièreté de SPRINTIA.
  • Votre temps total d'entraînement sur le graphique nommé "Entraînement récent (3 sem.)" est désormais arrondi pour éviter de se retrouver avec 185,25 min.
  • Correction d'un bug dans l'historique d'entraînement qui déclenchait le logo dynamique lors d'un clic sur le bouton "Afficher plus".
  • Correction d'un bug qui empêchait d'enregistrer un entraînement de 23h59m59s.

🗑️ Fonctionnalités supprimées

  • Les outils : "Estimation puissance moy. en ski" et "Estimation puissance max. en course" ont été supprimés car il y avait aucune étude scientifique sur les méthodes de calcul que j'utilisais.
  • La bannière "SPRINTIA est conçue pour vous aider avant et après..." a été supprimée de la page d'accueil.
  • La fonctionnalité nommé "Harmonie" pour les thèmes a été supprimée.
