SPRINTIA 4.0.3 : Préparation

Information

Type : Patch majeur
Date de sortie : 15 Mars 2026
Taille du site : 2,44 Mo

SPRINTIA 4.0.3 apporte des optimisations et des corrections de bugs. De plus, vous pouvez désormais voir vos statistiques d'un entraînement dans une page Web dédiée. Cette fonctionnalité a pour but de préparer SPRINTIA 4.1 qui ajoutera de nouveaux sports.

🆕 Nouvelles fonctionnalités

  • Nouveaux thèmes "Muguet" et "Citron".
  • Ajout de la possibilité de voir le détail de votre entraînement dans une page conçue pour ça.

📊 Améliorations

  • Désormais, dans l'historique d'entraînement la durée de l'entraînement s'affiche soit hh:mm:ss ou mm:ss.

🐛 Correction des bugs et optimisations

  • Changement des noms de class CSS pour améliorer la maintenabilité du code.
  • Ajout d'un conteneur autour de l'onglet sélectionné dans la navbar pour la version ordinateur.
  • Ajout d'une fonction globale qui convertit la durée (min) en hh:mm:ss ou mm:ss, ça permet d'optimiser le code de SPRINTIA.
  • Correction d'un bug qui permettait de modifier un entraînement qui était supprimé de la base de données.
