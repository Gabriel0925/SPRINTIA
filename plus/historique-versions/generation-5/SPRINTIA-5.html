Résumé

SPRINTIA 5 est une refonte majeure, l'application est désormais organisée et structurée ce qui améliore grandement l'expérience utilisateur. De plus, SPRINTIA 5 adopte enfin une tab-bar pour mobile, ainsi que du Glassmorphism pour accompagner le Glow Effect qui a été intégré dans SPRINTIA 4.3.

Anecdote

Il y a un an à quelques jours près je sortais la toute première version de SPRINTIA. Un an plus tard, je sors la 5ème version majeure de SPRINTIA et je remarque que le projet a bien grandi, bien plus que je ne l'imaginais !

Patch note

Nouvelles fonctionnalités

  • Ajout d'une tab-bar pour la version mobile ainsi que certaine tablette. Cette tab-bar réinvente totalement la navigation dans l'application, aux yeux de l'utilisateur ça améliore l'expérience car les boutons de navigation sont accessibles en permanence sans aucun clique contrairement au menu burger de l'ancienne version.
  • La structure de l'application a été totalement revue ! Le header ainsi que la tab-bar propose désormais 4 onglets : Entraînement, Progression, Outils et Paramètres. Tout est enfin ranger à sa place, ce qui est nettement mieux pour les utilisateurs ainsi que pour mon côté maniaque.
  • Le logo dynamique bien que plus présent de première abord est toujours présent, il est juste cacher, il intervient quand il le faut avec une toute nouvelle animation plus expressive et plus optimisé que l'ancienne.

Améliorations

  • Les boutons flottants s'accompagnent désormais du glassmorphism pour laisser transparaître le contenu en dessous tout en gardant une excellente lisibilité.
  • Désormais, SPRINTIA a été enfin optimisé pour les grands écrans, ce qui évite d'avoir des grosses bandes noires sur les côtés lors de la navigation.
  • La structure du header a été totalement revue, plus compact, plus joli grâce au glassmorphism. De nouvelles animations font leurs apparitions également.
  • Dans les paramètres, il y a désormais un bouton qui permet d'accéder directement à la page d'historique des versions de SPRINTIA. Ce changement a été réalisé car auparavant il fallait aller au bout du carrousel pour accéder à cette page, ce qui était un peu caché et pas très intuitif pour les utilisateurs.
  • Ajout d'icônes dans la page d'import de données ce qui est vachement plus joli.
  • L'interface dans les thèmes a très légèrement changé, surtout sur le thème qui est séléctionné.
  • Nouveau design de la page "Outils" pour améliorer l'UX et éviter de surcharger l'interface.
  • Nouveau design pour les blocs de résultats dans les outils car auparavant c'était des blocs de couleurs alors que ce n'était pas cliquable ce qui pouvait porter à confusion, c'est pour cela qu'ils adoptent désormais un nouveau design.
  • L'outils nommé "Estimation du temps de récupération" a été renommé "Temps de récupération".
  • Désormais, SPRINTIA ne s'écrit plus comme ça "SPRINTIA" mais comme ça "SPRINTIA", l'objectif est de rendre le nom de l'app un peu plus expressif.
  • La page des thèmes bénéficie enfin d'un vraie responsible, ce qui permet d'avoir une interface conçue pour le format smartphone, tablette, ordinateur,...
  • Le bouton "Calculer" dans l'outil "Protéines quotidiennes" a été supprimé, désormais l'outil calcule tout en temps réel.
  • Meilleure gestion du Glow Effect pour la version tablette et ordinateur.

Correction des bugs et optimisations

  • Optimisation des animations CSS pour gagner en performance et éviter de consommé des ressources inutilement : passage de "all" à par exemple "transform".
  • Désormais, dans de nombreux boutons il est désormais impossible de séléctionner le texte à l'intérieur du bouton.
  • Optimisation des liens pour les différents étages de l'arborecence de SPRINTIA, désormais au lieu de glisser des liens sur chaque page, j'ai mis en place une fonction JS qui permet de gérér les liens en fonction de la profondeur de la page.
  • Optimisation de nombreuses interfaces pour devenir intuitif peut importe la taille de l'écran.
  • Une nouvelle variable de temps pour les transitions fait son apparition, elle est utilisé lors du moment active c'est à dire lorsque l'utilisateur clique sur un élément de l'interface, ce qui donne un rendu plus réactif car l'animation est désormais 2x plus rapide.
  • Correction d'un bug d'affichage dans la page de statistiques qui pouvait laisser des anciennes datas lors du changement de période.
  • Correction d'un bug du graphique dans la page de statisitiques : lorsque l'utilisateur n'avait pas de données sur les 7 derniers jours, puis qu'il allait sur les 30 derniers jours, le graphique apparaissait donc, sauf que lorsque l'utilisateur revient sur les 7 derniers jours, le graphique restait affiché alors qu'il n'y avait pas de données, c'est désormais résolu.
  • La licence et la politique de confidentialité sont désormais accessible depuis les paramètres étant donné que le footer a été supprimé.
  • Changement de l'icone "Importer des données" pour plus de clarté aux yeux de l'utilisateur.
  • Optimisation du code de l'outil nommé "Protéines quotidiennes".
  • Ajout de plusieurs icônes dans les menus flottants comme par exemple dans le menu plus dans la page de détail de votre entrainement.
  • Correction d'un bug qui empêchait de pouvoir cliquer sur le contenu qui défilait entre le logo dynamique et les boutons de navigation ou du bouton burger-menu.
  • Correction d'un bug d'interface sur ordinateur, ce bug est connu depuis longtemps, il est désormais réglé : le bouton flottant n'était pas centrer quand votre écran était supérieur à 1250px, désormais il est bien centré.

Fonctionnalités supprimées

  • Le SPRINTIA Labs a été supprimé car je pensais m'en servir mais finalement je ne m'en suis pas servi en plus d'un mois, j'ai donc pris la descision de le supprimer.
