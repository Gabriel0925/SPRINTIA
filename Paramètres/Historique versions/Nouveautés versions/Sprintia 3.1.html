Sprintia 3.1 - Septembre 2025

Information

Type : Mise à jour mineure
Date de sortie : 31 Août 2025

Un nouvelle algorithme nommé "Indulgence de course" fais son apparition dans Sprintia 3.1. Cet algorithme va vous aider à ajuster votre kilométrage des 7 derniers jours pour rester dans une progression optimale et réduire le risque de blessure.

🆕 Nouvelles fonctionnalités

  • Une sidebar apparait sur la gauche pour naviguer plus facilement entre les différentes sections de Sprintia.
  • Option “Tous” : affiche désormais l’historique complet des entraînements peut importe le mode d'entraînement choisi.
  • "Quoi de neuf dans Sprintia 3.1" : nouvelle section dans les paramètres (accessible directement dans l’app).
  • Interprétation de l’IMC : ajout d’une phrase explicative en plus de l'IMC.
  • Estimation de votre VMA : calcul et affiche vos zones d'allures basées sur votre VMA estimée.
  • Nouveau statut “Pas commencé” pour les objectifs.
  • Nouvelle raison “Suspendre” pour la mise en pause des analyses.
  • Nouvelle donnée “Score” pour le football.
  • Nouveau mode "Course" pour l'enregistrement de données spécifique à la course à pied, au trail,...
  • Nouvelle algorithme "Indulgence de course" : ajuste le kilométrage des 7 derniers jours pour une progression optimale.
  • Modification complète d’un objectif ou d’une compétition via un seul bouton (date, lieu, statut…).

📊 Améliorations

  • Filtres persistants : mémorisation des choix (ex. conserver “1 mois” lors du passage d’Extérieur à Musculation).
  • Conseils et interprétations enrichis dans Charge d’entraînement.
  • Zones de fréquence cardiaque : affiche désormais la fréquence cardiaque maximale (FCM) dans le tableau des zones.
  • Estimation VO₂max : ajout d’une interprétation des zones plus détaillée.
  • Aide intégrée : explications sur le diverses choses lors de la première utilisation de Sprintia.
  • Tableaux réorganisés pour plus de cohérence.
  • Affichage d'un tiret en cas d'absence de données dans les tableaux (au lieu de cases vides).
  • Nettoyage de la base de données pour plus de rapidité.
  • Possibilité de mettre un autre type de pause même si une pause est déjà active (ex : passer de 'blessure' à 'vacances').
  • Programme bêta disponible depuis les paramètres.
  • Sprintia est disponible sur GitHub.
  • Après l’enregistrement d’une activité, d'un objectif ou d'une compétition, tu reviens automatiquement à la page précédente pour gagner du temps.

🐛 Correction des bugs et optimisations

  • Bio trop longue : problème résolu dans le profil et la modification de profil.
  • Coupure visuelle corrigée dans les en-têtes et le nom d’historique d’activité (Extérieur).
  • Coupure visuelle corrigée lorsque que le nom d'entraînement est long lors de la supression d'une activité.
  • Diverses corrections de fautes d'orthographe dans l'application.
  • Sprintia va désormais te tutoyer pour être plus proche de son utilisateur.
  • Gestion des erreurs du format de la date à l'ajout d'un objectif et d'une compétition.
  • Correction de bug lors de la suppression d'un entraînement.
  • Sécurité des données améliorée.
  • Optimisations du code pour la sidebar.
