Importer des données

Garmin

Vous avez une montre Garmin et vous voulez importer votre historique d'entraînement sur Sprintia, vous êtes au bon endroit. Pour importer vos données Garmin, veuillez télécharger le fichier CSV de votre historique d'entraînement disponible sur Garmin Connect Web. Pour récupérer ce fichier, suivez les instructions :

  • Cliquez sur le lien ci-contre.
  • Connectez-vous à votre compte Garmin.
  • La langue de Garmin Connect Web doit être le Français ou l'Anglais.
  • Cliquez sur "Export CSV" (en haut à droite), le fichier CSV a dû se télécharger.

Conseil

  • 1er conseil : Importer uniquement les 30 derniers jours de votre historique d'entraînement, juste pour calibrer les algorithmes de Sprintia.
  • 2ème conseil : Vérifier dans l'historique d'entraînement de Sprintia que tout a bien été importer correctement. Si ce n'est pas le cas, je vous conseille de modifier l'entraînement pour corriger l'erreur.
