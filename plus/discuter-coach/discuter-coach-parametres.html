Paramètres discussion

Lorsque vous discutez avec votre coach, SPRINTIA crée un prompt avec :

  • Vos données récentes (pour donner le contexte à l'IA)
  • Votre question
  • Des instructions pour personnaliser l'expérience de discussion entre votre IA favorite et vous

Le problème c'est que quand vous analysez vos tendances, votre entraînement ou votre récupération SPRINTIA ne fournit pas les données de votre profil (sexe, âge, taille, poids, FC repos) mais quand vous discutez avec le coach, SPRINTIA fournit dans le prompt toutes vos données récentes d'entraînement, de récupération et également de votre profil.

Or vous n'avez peut-être pas envie que Gemini (ou ChatGPT ou autre IA) connaisse votre poids, votre taille,... ce qui est totalement compréhensible c'est pour ça que vous pouvez désactiver le partage de vos données de profil (sexe, âge, taille, poids, FC repos) dans le prompt que SPRINTIA génére.

Partager mon profil

En activant la fonctionnalité ci-dessus, SPRINTIA ajoutera les données de votre profil dans le prompt qu'il générera ce qui permettra à l'IA d'être plus précise car le prompt lui donnera davantage de contexte mais ce n'est clairement pas obligatoire si vous ne souhaitez pas partager les données de votre profil.
