Sprintia 3.1.1 : Corrections mineures

Information

Type : Patch majeur
Date de sortie : 03 Septembre 2025

Cette mise à jour apporte des correctifs de bugs et des améliorations diverses.

📊 Améliorations

  • Migration automatique de la base de données : le processus de mise à jour de la base de données est maintenant géré automatiquement, quelle que soit la version de départ de votre installation. Les utilisateurs des versions 2.0 et 3.0 peuvent désormais mettre à jour directement vers la version 3.1.1 sans manipulation supplémentaire.
  • La valeur par défaut du RPE est 5 ce qui évite de provoquer un décalage dans la charge d'entraîenement.

🐛 Correction des bugs et optimisations

  • Correction du format de date dans l’historique pour les modes Course, Musculation et Football on passe au format JJ/MM/AAAA.
  • Correction de plusieurs textes coupés dans les interprétations de la charge d'entraînement.
  • Correction du bug bloquant les changements de statut de pause : il est désormais possible de passer directement d’un statut à un autre (ex. : de blessure à vacances, de suspendre à blessure,...) sans bug.
