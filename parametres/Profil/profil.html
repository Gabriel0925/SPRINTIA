Sauvegarde

Vous voulez changer de smartphone ou alors vous souhaitez transférer vos données vers un autre appareil informatique (ex: votre PC) ? Vous êtes au bon endroit ! La fonctionnalité de sauvegarde permet de télécharger, exporter vos données au format JSON ce qui permettera par la suite de restaurer vos données (voir plus bas dans la page) sur votre nouvel appareil,...

Restauration

La fonctionnalité de restauration permet d'ingérer vos données (qui ont été téléchargée au format JSON au préalable) dans Sprintia. Cette fonction est utile si vous changez d'appareil informatique. Attention, en important des données vous écraserez vos données actuelle.
