Politique de confidentialité

Dernière mise à jour de la politique de confidentialité : 12 Février 2026

Information

Le but de Sprintia n'est pas de collecter des données, mais de vous aider à optimiser vos entraînements. C'est pour cela que Sprintia reste transparent sur la collecte de vos données ou même l'enregistrement dans la base de données de votre navigateur.

Données collectées

Je ne collecte aucune données personnelles dans votre dos. Lorsque vous utilisez les outils de Sprintia, la plupart n'ont pas besoin de données (ce sont des outils instantanés), mais pour certains outils comme la charge d'entraînement, Sprintia a besoin de votre historique d'entraînement pour calculer votre charge d'entraînement. Cependant, ces données sont enregistrées/traitées en local dans votre navigateur. Donc, vos données restent sur votre appareil, elles ne traversent pas les serveurs du monde entier,...

Sécurité

Si vous venez de lire toutes ces informations, vous avez compris que Sprintia stocke les données dans votre navigateur.
Donc, si vous effacez le cache de votre navigateur ou que vous réinitialisez votre appareil, Sprintia aura perdu la mémoire car vos données auront été entièrement supprimées. Vos données vous appartient, elles sont sous votre responsabilité.
Le point positif c'est que vous n'avez pas besoin de créer un compte pour utiliser Sprintia, mais cela signifie que les données que vous enregistrez sont uniquement disponibles sur votre appareil.
Si vous utilisez Sprintia au quotidien avec votre téléphone, que vous avez même choisi un thème,..., si vous allez sur Sprintia depuis votre PC, vous ne pourrez pas retrouver les données que vous avez sur votre smartphone.

Sauvegarde

Imaginons que vous changiez de smartphone. Vous aimeriez transférer vos données de l'ancien téléphone vers votre nouveau. Ou alors de votre téléphone à votre PC pour une raison quelconque. Grâce à Sprintia, vous pouvez enregistrer toutes vos données dans un fichier JSON.
Après, vous pourrez transférer ce fichier de votre ancien appareil vers votre nouveau. Et la dernière étape sera de l'importer dans Sprintia en cliquant sur "Restaurer mes données".

Cookies

N'importe quelle personne qui s'est déjà promenée sur internet a pu voir qu'un site lui demande d'accepter les cookies.
En réalité, les cookies ne sont pas faits pour être mangés sur internet, mais pour vous indiquer que le site va stocker des données plus ou moins personnelles (ça dépend des cas). C'est pour cela que, pour le moment, je ne préfère pas mettre des cookies sur Sprintia.
