Sprintia 4.0.2 : Corrections majeures

Information

Type : Patch majeur
Date de sortie : 07 Mars 2026
Taille du site : ... Mo

Sprintia 4.0.2 apporte de nombreuses optimisations et correction de bug ce qui améliore la stabilité de Sprintia. De plus, Sprintia 4.0.1 vous offre la possibilité d'importer votre historique d'entraînement depuis un fichier CSV qui provient de Garmin Connect (Web), TrainingPeaks ou encore Runkeeper.

🆕 Nouvelles fonctionnalités

  • Vous pouvez désormais importer la totalité de votre historique d'entraînement depuis un fichier csv sur le site de Garmin, TrainingPeaks ou encore Runkeeper.

📊 Améliorations

  • Changement de design pour les listes à puce.
  • 2 nouveaux avatar de coach sont disponible : "🏃", "🚴".
  • Nouvelle interface dans la page d'accueil, avec un bouton "ajouter" repensé et un statut plus mis en avant.
  • Il y a de nouveau l'effet de rebond lors du clic sur une cartes.
  • Une nouvelle licence fais son apparition ce qui protège mon code pour éviter que quelqu'un utilise mon code a but commercial ou pour créer une société. De plus, un utilisateur n'a pas le droit de prendre mon code, de changer le logo, et de le remettre en ligne sous son propre nom (même gratuitement).

🐛 Correction des bugs et optimisations

  • Correction d'un bug qui empêchait le calcul de vos Zones cardiaques avec la méthode FC max.
  • Correction d'un bug qui empêchait l'affichage de certaines icônes dans les cartes des outils.
  • Correction d'un bug qui permettait d'afficher 24.000000033300033030min dans le JRM Coach à la page d'accueil.
  • La distance est désormais arrondi dans l'historique d'entraînement.
  • Correction d'un bug de la charge d'entraînement qui bloquait les analyses si vous n'aviez pas rentrer d'entraînement depuis 7 jours.
  • Les textes à l'intérieur des boutons quand une action se lance ont été améliorés.
  • Correction d'un bug très mineur dans l'outil : "Niveau de course".
  • Les coins sont moins arrondit et désormais il respecte une loi d'UI pour les cartes.
  • Le code couleur des messages d'erreur dans l'outil Niveau de Course respecte désormais le code couleur imposé par la mise à jour Sprintia 4.0.1.
  • Grosse optimisation du code CSS avec de meilleur nom de class,...

🗑️ Fonctionnalités supprimées

  • Les outils : "Estimation puissance moy. en ski" et "Estimation puissance max. en course" ont été supprimés car il y avait aucune étude scientifique sur les méthodes de calcul que j'utilisais.
  • La banière "Sprintia est conçue pour vous aider avant et après..." a été supprimé de la page d'accueil.
  • La fonctionnalité nommé "Harmonie" pour les thèmes a été supprimé.
