Importer un fichier TrainingPeaks CSV

Vous avez synchronisé votre montre (Apple Watch, Coros, Polar, Suunto...) avec TrainingPeaks. Vous pouvez grâce à cela importer votre historique d'entraînement sur SPRINTIA en important le fichier '.csv' que vous pouvez trouver sur le site web de TrainingPeaks. Pour ce faire, suivez les instructions :

  • Cliquez sur le lien ci-contre.
  • Connectez-vous à votre compte TrainingPeaks.
  • Allez dans les paramètres (cliquer sur votre photo de profil en haut à droite puis "Settings").
  • Rendez-vous dans "Export data".
  • Remplissez les champs de la catégorie "Workout Summary".
  • Appuyez sur "Exporter".

Pour finir, vous n'avez plus qu'à dézipper le dossier et importer le fichier CSV nommé 'workouts.csv' dans SPRINTIA.

Conseil pour l'importation

  • 1er conseil : Importer uniquement les 30 derniers jours de votre historique d'entraînement, juste pour calibrer les algorithmes de SPRINTIA.
  • 2ème conseil : Vérifier dans l'historique d'entraînement de SPRINTIA que tout a bien été importer correctement. Si ce n'est pas le cas, je vous conseille de modifier l'entraînement pour corriger l'erreur.
